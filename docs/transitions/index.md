# Transition Presets

Alight Motion includes **25 transition presets** — pre-configured effect setups that animate layers on or off screen.

Transition presets use combinations of existing effects (Wipe, Dissolve, Lightning, etc.) with specific parameter settings to create smooth scene transitions. They are found in the **Presets** panel when editing an effect.

## How Transitions Work

Transition presets are pre-configured **effect presets** — they bundle one or more effects with:

- **Keyframed parameters** — values animated from start to end over time
- **Specific effect choices** — tailored for each transition style (wipe, glow, spin, etc.)
- **Suggested duration** — typically 0.5–1.5 seconds

To use a transition preset:

1. Select the layer you want to transition
2. Tap **Effects → Presets**
3. Choose **Transition In** or **Transition Out**
4. Pick a style and adjust timing as needed
