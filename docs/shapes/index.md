# Shape Templates

Alight Motion includes **20 built-in shape templates**. Each shape is defined by an XML template with parameters and JavaScript path-generation logic.

> Learn how shapes work in the [Getting Started Guide](/guide#shape-templates). Each shape page includes a rendered SVG preview showing its default appearance.

- ![Arc](/shapes/arc.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Arc](/shapes/arc)** — A curved line segment defined by start angle, end angle, and radius.
- ![Arrow](/shapes/arrow.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Arrow](/shapes/arrow)** — A line with an arrowhead at one end.
- ![Callout](/shapes/calloutrr.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Callout](/shapes/calloutrr)** — A rounded rectangle speech bubble / callout shape.
- ![Circle](/shapes/circle.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Circle](/shapes/circle)** — A perfect ellipse (oval or circle).
- ![Line](/shapes/line.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Line](/shapes/line)** — A straight line segment between two points.
- ![Moon](/shapes/moon.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Moon](/shapes/moon)** — A crescent moon shape.
- ![Multifoil](/shapes/multifoil.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Multifoil](/shapes/multifoil)** — A multi-lobed flower / clover shape (trefoil, quatrefoil, etc.).
- ![Irregular Pentagon](/shapes/penta.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Irregular Pentagon](/shapes/penta)** — A pentagram / star polygon.
- ![Pie](/shapes/pie.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Pie](/shapes/pie)** — A pie/wedge shape (like a pizza slice).
- ![Plus](/shapes/plus.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Plus](/shapes/plus)** — A plus/cross shape.
- ![Polygon](/shapes/poly.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Polygon](/shapes/poly)** — A regular polygon with configurable side count.
- ![Quad](/shapes/quad.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Quad](/shapes/quad)** — A quadrilateral with four adjustable corners.
- ![Rectangle](/shapes/rect.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Rectangle](/shapes/rect)** — A rectangle with adjustable width and height.
- ![Rounded Rectangle](/shapes/roundrect.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Rounded Rectangle](/shapes/roundrect)** — A rounded rectangle with corner radius control.
- ![Stamp](/shapes/stamp.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Stamp](/shapes/stamp)** — A multi-purpose stamp shape with configurable lobes and indentation.
- ![Star](/shapes/star.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Star](/shapes/star)** — A star shape with configurable point count and inner/outer radius.
- ![Teardrop](/shapes/teardrop.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Teardrop](/shapes/teardrop)** — A teardrop / droplet shape.
- ![testshape](/shapes/testshape.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[testshape](/shapes/testshape)** — A simple test shape for debugging.
- ![Triangle](/shapes/triangle.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Triangle](/shapes/triangle)** — An equilateral triangle.
- ![Wide Line](/shapes/wideline.svg){style="width:48px;height:48px;vertical-align:middle;margin-right:8px"} **[Wide Line](/shapes/wideline)** — A wide rectangular line / bar.
